HOW TO MAKE A LIST WITH HTML






If you want to make a list in a website with html there's a simple tag for it


There's 2 lists types: ordered list and numbered list


The ordered list tag is:
<"ol">
<"li"> <"/li">
<"ol">



The numbered list tag is:
<"ul">
<"li"> <"/li">
<"ul">
